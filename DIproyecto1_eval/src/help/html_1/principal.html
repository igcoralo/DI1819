Ayuda aplicación carreras

La aplicación está destinada al almacenamiento y ejecución de una seríe de carreras y al almacenamiento de una base de datos de corredores

Esta página es la ayuda principal. La aplicación consta de varias ventanas, la ayuda a los pasos más complejos se encuentra en las siguientes dos ventanas, una corredores y la otra carreras. Cada una de las opciones de la página principal tiene su propia página de ayuda.
